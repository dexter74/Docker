### Guide ####
````
https://doc.traefik.io/traefik/routing/overview/
- Providers   : Les découvrent les services qui vivent de votre infrastructure (ip, santé, ...)
- Entrypoints : Écoutent le trafic entrant (ports, ...)
- Routeurs    : Analysent les demandes (hôte, chemin, en-têtes, SSL, ...)
- Services    : Transmettre la demande à vos services (équilibrage de charge, ...)
- Middlewares : Peut mettre à jour la demande ou prendre des décisions en fonction de la demande (authentification, limitation de taux, en-têtes, ...)
````
